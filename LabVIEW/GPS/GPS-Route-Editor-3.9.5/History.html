  1. Native x64 지원:

  2. 위치찾기 기능 추가:
    * 주소 또는 지명으로 장소를 검색하여 이동 가능.
    * 검색한 지역을 웨이포인트로 바로 저장 가능.

    * Waypoint의 컨텍스트 메뉴(마우스 오른쪽 클릭 -> 'MYP 파일로 저장')에서 개별 저장 가능.
    * Elements 창의 최상위 항목인 Waypoints의 컨텍스트 메뉴(마우스 오른쪽 클릭 -> 'MYP 파일로 모두 저장')에서 일괄 저장 가능.
  6. 처음 프로그램을 설치하여 실행할 때, 오류가 발생하는 문제 수정.
    * 이전 버전 사용자의 경우 해당 오류가 발생하지 않음.
  7. 외부에서 불러온(Import) GPX 파일에 고도(Altitude; Elevation) 정보가 있을 경우, 해당 정보를 보존하도록 변경.
    * 향후 버전에서 고도 정보를 처리하기 위한 준비 과정입니다.
  8. 그 외 사소한 업데이트.
